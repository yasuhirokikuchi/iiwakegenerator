# 言い訳ジェネレーター (Excuse Generator)

AI（Gemini）を活用して、角を立てずに誘いや依頼を断るための「言い訳」を自動生成するデスクトップアプリケーションです。

## 特徴

- **多様なテイスト**: 丁寧なビジネス向けから、武士風、ギャル風、意識高い系まで、シチュエーションに合わせて選択可能。
- **Gemini AI 搭載**: Google の最新 AI モデル（Gemini 3.5 Flash）が、文脈に応じた最適な言い訳を作成します。
- **ワンクリックコピー**: 生成された文章をすぐにクリップボードにコピーして利用できます。
- **直感的な UI**: シンプルでモダンなチャット風インターフェース。

## 動作要件

- Node.js (LTS 推奨)
- Google AI Studio (Gemini) API キー

## セットアップ

1. **リポジトリをクローンまたはダウンロードします。**

2. **依存関係をインストールします。**

   ```bash
   npm install
   ```

3. **環境変数の設定**
   ルートディレクトリに `.env` ファイルを作成し、Gemini の API キーを設定してください。
   ```env
   GEMINI_API_KEY=あなたのAPIキーをここに記入
   ```
   ※ API キーは [Google AI Studio](https://aistudio.google.com/) から取得できます。

## 使い方

1. **アプリを起動します。**

   ```bash
   npm start
   ```

2. **「断りたいこと」を入力します。**
   例：金曜日の飲み会、週末の残業、貸した本の返却など。

3. **「テイスト」を選択します。**
   - 丁寧（ビジネス用）
   - 武士風
   - ギャル風
   - 意識高い系

4. **「生成する」ボタンをクリックします。**

5. **生成されたメッセージを確認し、必要に応じて「コピー」ボタンでコピーして使用します。**
